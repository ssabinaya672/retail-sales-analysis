# Retail Sales End-to-End Data Analysis

## Overview
A complete data analysis project using Python, SQL and Power BI
to analyze 9,800+ retail sales records from 2015-2018.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL (SQLite)
- Power BI

## Key Findings
- Total Sales: $2,261,536
- Top Category: Technology ($827,455)
- Top Region: West ($710,219)
- Top State: California ($446,306)
- Sales peak in Q4 (October - December)

## Project Structure
- sales_analysis.ipynb → Python analysis & visualizations
- sales_queries.sql → SQL business queries
- sales_dashboard.pbix → Power BI interactive dashboard
- train.csv → Dataset
